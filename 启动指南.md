# MarketNiche MVP 启动指南

## 🎯 快速启动（5分钟）

### 第一步：启动后端

```bash
# 进入后端目录
cd backend

# 安装依赖
pip install -r requirements.txt

# 初始化数据库（创建表并插入示例数据）
python init_db.py

# 启动后端服务
uvicorn app.main:app --reload
```

后端将在 **http://localhost:8000** 启动

✅ 验证：访问 http://localhost:8000/docs 查看API文档

### 第二步：启动前端

打开新的终端窗口：

```bash
# 进入前端目录
cd frontend

# 安装依赖（首次运行）
npm install

# 启动前端开发服务器
npm run dev
```

前端将在 **http://localhost:5173** 启动

✅ 验证：浏览器访问 http://localhost:5173

## 🔧 常见问题

### 问题1：pip install 失败
**解决**：使用虚拟环境
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
pip install -r requirements.txt
```

### 问题2：npm install 失败
**解决**：清除缓存后重试
```bash
npm cache clean --force
npm install
```

### 问题3：数据库初始化失败

# 确保backend目录有写入权限

### 问题4：端口被占用
**解决**：修改端口
- 后端：修改 `uvicorn app.main:app --reload --port 8001`
- 前端：修改 `vite.config.ts` 中的 `server.port`
